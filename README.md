## Code setup
#### First install all the dependencies
In the project directory, run:
##### `npm install`
Then
##### `npm start`

Runs the app in the development mode.

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
## Deployment
In the project directory, run:
##### `npm run deploy`
